# Análise de Consumo Musical: Springfield vs. Shelbyville 🎧

Este projeto analisa o comportamento de consumo musical em duas cidades distintas, utilizando dados reais de streaming para identificar padrões de comportamento temporal e preferências de gênero.

## 🚀 Objetivo
O objetivo principal foi realizar o processamento completo dos dados (limpeza, transformação e análise) para entender se os gostos musicais divergem entre as cidades e como o volume de reproduções varia ao longo da semana.

## 🛠️ Tecnologias Utilizadas
* **Python 3**
* **Pandas:** Manipulação e limpeza de dados.
* **Matplotlib:** Visualização de dados e criação de gráficos.
* **Jupyter Notebook:** Ambiente de desenvolvimento.

## 📊 Principais Entregas Técnicas
* **Pipeline de Limpeza:** Tratamento de valores ausentes e remoção de duplicatas para garantir a integridade da análise.
* **Funções de Automação:** Criação de funções modulares para filtragem de dados e geração automática de gráficos profissionais.
* **Insights de Negócio:** Identificação de picos de consumo específicos (segunda/sexta em Springfield vs. quarta-feira em Shelbyville).

## 📂 Como Executar
1. Clone este repositório.
2. Certifique-se de ter o Python e a biblioteca Pandas instalados.
3. Abra o arquivo `music_analysis.ipynb` em seu ambiente Jupyter.
